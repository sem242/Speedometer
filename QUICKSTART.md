# 🚀 Quick Start - GPS Speedometer

## Stap 1: Start de Server

### Python (Aanbevolen - Snelst)
```bash
python server.py
```

### Node.js
```bash
npm install
npm start
```

## Stap 2: Open in Browser

**Desktop**: `http://localhost:8000`

**iPhone op zelfde WiFi**:
1. Vind je computer IP: Zet `ipconfig` in PowerShell
2. Open Safari op iPhone
3. Ga naar: `http://<IP>:8000` (bijv: `http://192.168.1.50:8000`)

## Stap 3: Gebruik

1. Tik "Start GPS"
2. Accepteer locatietoegang
3. Rijd en zie je snelheid!

## Tips

- ✅ Outdoor voor beste GPS
- ✅ iPhone Safari geeft beste nauwkeurigheid
- ✅ Voeg toe aan Home Screen voor app-achtig gebruik
- ✅ Geen internet nodig als gehost op lokaal netwerk
